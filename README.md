# Юридическая компания "Правовая Защита"

Сайт для юридической компании, специализирующейся на защите прав предпринимателей в спорах с ФНС.

## Структура проекта

- `index.html` - Главная страница
- `css/style.css` - Стили сайта
- `js/main.js` - JavaScript функционал
- `robots.txt` - Файл для поисковых роботов
- `sitemap.xml` - Карта сайта
- `README.md` - Документация

## Особенности

- Современный адаптивный дизайн
- SEO-оптимизированный контент
- Интерактивные элементы
- Мобильная навигация
- Форма обратной связи
- Расширенные метатеги для социальных сетей
- Анимации и карусели

## Расширенные метатеги

Сайт содержит полный набор метатегов для правильного отображения в:
- Facebook (Open Graph)
- Twitter (Twitter Cards)
- VKontakte
- Telegram
- WhatsApp
- Других мессенджерах

## Установка

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в браузере

## Используемые технологии

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6)
- Google Fonts
- Font Awesome Icons

## SEO оптимизация

- Правильные заголовки H1-H6
- Метатеги description и keywords
- Структурированные данные
- Карта сайта (sitemap.xml)
